実務で役立つWordの基本

文書作成から表・図形まで、初心者がまず身につけるべき操作を網羅

Microsoft Word（ワード）は、ビジネスにおける文書作成の標準ツールです。チラシ作りから報告書の作成まで幅広く使えますが、我流で使っていると、後からレイアウトが崩れて苦労することも少なくありません。

この記事では、現役パソコン講師の視点から、初心者がまず押さえるべき「Wordの正しい操作手順」を解説します。基本をしっかりマスターして、効率的な文書作成を目指しましょう。

💡 講師からのアドバイス：設定は「上から順番に」
Wordで文書を作る際、いきなり文字を打ち始めていませんか？ 実は、一番最初に「用紙の設定」を済ませることが、レイアウト崩れを防ぐ最大のコツです。後から用紙サイズや余白を変えると、すべての配置がずれて修正に時間がかかってしまいます。

ページレイアウトの基本設定

文書を作り始める前に、まず「用紙のサイズ・向き・余白」を設定しましょう。

設定項目 操作場所 よく使う値
用紙サイズ [レイアウト]タブ → 用紙サイズ A4（最も一般的）
用紙の向き [レイアウト]タブ → 印刷の向き 縦（通常）/ 横（図表が多い場合）
余白 [レイアウト]タブ → 余白 標準：上下25mm・左右30mm
実務のヒント：
ビジネス文書では「ユーザー設定の余白」から数値を微調整することがよくあります。情報を1枚に収めたい場合は少し狭くし、読みやすさを重視する場合は標準より広めに取るのがプロの工夫です。

文字の範囲選択と効率的な操作

文字の色や大きさを変えるには、まず「ここを変える」という範囲選択が必要です。マウス操作だけでなく、キーボードを併用すると格段に効率が上がります。

  • 1文字ずつ選択： Shift + ← / →
  • 単語を選択： 目的の単語をダブルクリック
  • 1行全体を選択： 行の左端（矢印が右を向く場所）をクリック
  • 文書全体を選択： Ctrl + A

文字の配置と見やすい装飾

文字の配置（左・中央・右）や強調の装飾は[ホーム]タブで行います。ショートカットキーを活用することで、マウスへ手を動かす手間を最小限に抑えられます。

主要なショートカットキー：
  • 太字： Ctrl + B
  • 中央揃え： Ctrl + E
  • 右揃え： Ctrl + R
  • 上書き保存： Ctrl + S

特に「両端揃え（Ctrl + J）」は、文書の右端をビシッと揃えてくれるため、ビジネス文書を整然と見せるのに非常に効果的です。

表の作成とレイアウト調整

スケジュール表や比較表など、データを整理して見せるために「表」は欠かせません。[挿入]タブから行数と列数を指定して作成します。

表を挿入すると、リボンに「テーブルデザイン」と「レイアウト」という2つの専用タブが現れます。ここでセルの結合や背景色の変更を行います。セルの配置を「中央揃え」に設定すると、表の中の文字が美しく収まります。

段落の書式と箇条書きの活用

読みやすい文書を作る秘訣は、適切な「余白」と「構造化」にあります。単に文字を並べるのではなく、Wordの機能を活用しましょう。

箇条書きを使用する際は、手入力で「・」を打つのではなく、必ず[ホーム]タブの「箇条書き」ボタンを使いましょう。これにより、2行目以降の文字の開始位置（インデント）が自動で揃い、非常に見やすくなります。

💡 講師からのアドバイス：編集記号を味方にする
[ホーム]タブにある「編集記号の表示/非表示」をONにしてみてください。普段は見えない「スペース（□）」や「段落記号（改行マーク）」が見えるようになります。レイアウトが思うように決まらない時、余計なスペースが入り込んでいないかを確認するのに役立ちます。

画像や図形による視覚的な表現

文字だけの文書よりも、適度に図表が入った文書の方が情報は伝わりやすくなります。

  • 画像の挿入： 画像を選択した後は「レイアウトオプション」から「四角形」に設定すると、周囲に文字が回り込み、自由に配置できるようになります。
  • 図形の作成： Shift を押しながらドラッグすることで、正円や正方形をきれいに描くことができます。
  • テキストボックス： 本文とは独立した位置に文字を配置したい場合に重宝します。

検索と置換による一括修正

数十ページの文書で特定の単語をすべて書き換えたい場合、1つずつ探すのは時間の無駄です。Ctrl + H で「置換」画面を開き、古い用語と新しい用語を入力して一括で変更しましょう。事務作業のスピードを上げる重要なテクニックです。

印刷プレビューとPDFでの書き出し

最後に、作成した文書は必ず印刷プレビュー（Ctrl + P）で確認してください。「文字が1行だけ次ページにはみ出している」といったミスを事前に防げます。

また、作成したファイルを取引先へ送る際は、内容が改変されず、どのデバイスでも同じ見た目で開ける「PDF形式」でエクスポートすることを推奨します。

Word基礎学習のポイント

  • 設定は「レイアウト → 入力 → 装飾」の順で行う
  • ショートカットキーを使い、タイピングとの連動性を高める
  • 表や箇条書き機能を使い、読み手の視線を誘導する
  • 最後に必ず印刷プレビューとPDF化を確認する
